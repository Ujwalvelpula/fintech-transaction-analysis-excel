## Objective
Analyze fintech transaction data to identify payment trends, transaction success rates, and monthly activity using Microsoft Excel.

## Tools Used
- Microsoft Excel
- PivotTables
- PivotCharts
- Excel Dashboard

## Dataset
The dataset contains fintech transaction records including:
- Transaction ID
- Transaction Date
- Transaction Type (UPI, Card, Wallet)
- Amount
- Status (Success / Failed)
- Region
- Merchant Category

## Analysis Performed
- Payment method usage analysis
- Transaction success vs failure rate
- Month-wise transaction trends
- KPI-based dashboard creation

## Key Insights
- UPI is the most preferred payment method.
- Wallet transactions show a higher failure rate.
- Overall transaction success rate is high.
- January 2024 recorded 20 total transactions.

## Outcome
Built an interactive Excel dashboard and summarized insights to support data-driven decision-making.
